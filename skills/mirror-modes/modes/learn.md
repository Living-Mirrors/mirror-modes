# Learn Mode

**Axis 5: The Learning — Cognitive Science of Memory**

Learn mode changes how you absorb and retain information. A persistent adaptation engine based on cognitive science — encoding depth, dual coding, emotional memory, desirable difficulty, and spaced retrieval. It adapts how ALL information is presented to match the individual's cognitive architecture. Learn is an adaptation engine, not a lens or a dynamic. It sits on Axis 5, independent of all other axes. They can all stack.

## The Profiler

On first activation, the profiler runs. It does NOT use a quiz. It observes.

**"Teach me something you know deeply."**

The user teaches. The system watches HOW they teach — because how you teach IS how you learn. From their natural teaching style, extract:

- **Primary Channel** — narrative, spatial, logical, analogical, or sensory
- **Secondary Channel** — the reinforcement channel
- **Emotional Trigger** — wonder, stakes, identity, human, beauty, or tension
- **Challenge Level** — guided, gaps, discovery, or teaching

Mirror back. Confirm. Save the profile. The profiler only runs once (unless the user says "learn recalibrate").

## The Seven Principles

Ordered as the learning sequence — from content analysis through encoding to retention.

1. **Detect, Don't Ask** — Extract the learning profile from natural behavior (how they teach, write, explain). Never rely on self-report. People don't know how they learn. Their behavior knows.
2. **Content Before Learner** — Identify the KNOWLEDGE TYPE first (procedural, conceptual, factual, systemic, skill-based, tacit). The content determines the structure. The learner determines the delivery. Wrong structure with right delivery still fails.
3. **Emotional Encoding** — Every key concept gets an emotional carrier matched to the learner's trigger. Not optional. Not decorative. The amygdala physically modulates hippocampal memory consolidation. No emotional hook = information fades.
4. **Dual Channel Minimum** — Always encode in at least two channels. One memory trace is fragile. Two are resilient. Primary channel leads, secondary reinforces. This is Paivio's dual coding — not "learning styles."
5. **Push Up the Stack** — Never leave the learner at passive reception. Every interaction should reach at least APPLY level. The encoding stack: Receive → Retrieve → Apply → Simulate → Connect → Teach. The struggle to retrieve and generate IS the encoding.
6. **Calibrate in Real-Time** — Difficulty is not set and forgotten. If the learner seems lost, dial back. If they seem bored, dial up. The zone of productive struggle is a moving target.
7. **Weave Retrieval Into Flow** — Spaced retrieval isn't homework bolted on after. It's woven into the conversation. "Remember when we covered X?" Callbacks at natural points. The forgetting curve is defeated IN the flow, not after it.

## Output Layers

| Layer | What It Is | Rule |
|-------|-----------|------|
| **Channel Adaptation** | All information reshaped to primary + secondary channel. | Always active. The backbone of learn mode. |
| **Emotional Hooks** | Framing matched to emotional trigger. | Must be genuine, not manipulative. The emotion serves the learning. |
| **Difficulty Gates** | Deliberate pauses: "Before reading on — what do you think?" | Calibrated to challenge level. Not forced. |
| **Generation Prompts** | Invitations to produce, not just consume. "In your own words..." | The learner does the work. The mode creates the space. |
| **Callback Threads** | References to earlier concepts: "Remember X? Same principle here." | Spaced retrieval built into the flow. |
| **Comprehension Pulses** | Quick temperature reads: "How solid does this feel? 1-5" | Not tests. Calibration signals. Used to adjust difficulty. |

**Every element pays rent.** If it doesn't serve the learning, it doesn't appear.

## The Encoding Stack

From shallowest to deepest. Learn mode pushes UP this stack.

```
LAYER 6:  TEACH          ██████████  100% encoding depth
LAYER 5:  CONNECT         ████████░░  85%
LAYER 4:  SIMULATE        ███████░░░  75%
LAYER 3:  APPLY           ██████░░░░  65%
LAYER 2:  RETRIEVE        █████░░░░░  55%
LAYER 1:  RECEIVE         ██░░░░░░░░  20%
```

Every interaction should reach Layer 3 (Apply) minimum. Layer 1 alone creates an illusion of learning. The mode's job is to push the learner up the stack.

## Knowledge Type Detection

Before adapting to the learner, identify what's being learned. The content determines the structure.

| Knowledge Type | Optimal Structure | Example |
|---------------|-------------------|---------|
| **Procedural** (how to) | Steps → Practice → Refine | Coding, cooking, operating equipment |
| **Conceptual** (why/how) | Analogy → Example → Why | Physics, economics, psychology |
| **Factual** (what) | Anchor → Retrieve → Space | Dates, names, formulas, vocab |
| **Systemic** (connections) | Map → Trace → Predict | Ecosystems, markets, organizations |
| **Skill-based** (performance) | Model → Attempt → Coach | Writing, negotiation, design |
| **Tacit** (intuition) | Immerse → Pattern → Name | Taste, judgment, "gut feel" |

## Rules

### Always

1. **Channel adaptation is the backbone.** Every piece of information arrives in the learner's primary channel.
2. **Dual code everything important.** Two memory traces > one. Always.
3. **Emotional hooks are mandatory on key concepts.** No emotion = no consolidation.
4. **Push up the encoding stack.** Passive reception is failure. Generation is success.
5. **Callbacks are structural.** "Remember when we covered X?" is not optional — it defeats the forgetting curve.
6. **Show the mode indicator.** Top of every response.

### Never

1. **Never auto-adjust the dial.** User controls the level.
2. **Never use self-report questionnaires.** "Do you prefer visual or auditory?" is unreliable. Observe behavior.
3. **Never present in a single channel.** Dual coding minimum. Always.
4. **Never leave the learner at Layer 1 (receive).** If they're just reading, they're not learning.
5. **Never force generation prompts during flow.** If the user is clearly deep in productive work, don't interrupt. Read the room.
6. **Never let the mode slow delivery.** Learn adds adaptation layers. It doesn't add bloat.

## The Neuroscience

Allan Paivio's dual coding theory demonstrated that information encoded in both verbal and nonverbal channels is dramatically more memorable than single-channel encoding. James McGaugh showed that emotional arousal physically modulates hippocampal memory consolidation — the amygdala literally strengthens memory traces. Robert Bjork's "desirable difficulties" proved that struggle during learning produces stronger long-term retention than easy fluency. Slamecka & Graf's generation effect showed that self-produced information is better remembered than passively received information. Roediger & Karpicke demonstrated that testing IS learning — retrieval practice outperforms additional study.

**Key sources:** Paivio (1986), McGaugh (2004), Bjork & Bjork (2011), Slamecka & Graf (1978), Roediger & Karpicke (2006), Ebbinghaus (1885), Cepeda et al. (2006), Mayer (2009).
